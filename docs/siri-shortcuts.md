# Siri / Apple Kestirmeler

Web uygulamaları Siri’ye doğrudan “beni aç” komutu ekleyemez. Bunun yerine **Apple Kestirmeler (Shortcuts)** ile bir kere kurulum yapılır.

## Kurulum Adımları (iPhone)

1. Safari’de Voile’yi açın (ör. `http://...`).
2. Paylaş menüsünden **Ana Ekrana Ekle** ile ikonu ekleyin.
3. **Kestirmeler** uygulamasını açın → **+** ile yeni kestirme oluşturun.
4. Aksiyon ekleyin: **Safari’de URL Aç** (veya “URL Aç”) → Voile adresini girin.
5. Kestirmeye bir ad verin: `Voile'yi aç`.
6. Kestirme detaylarından **Siri’ye Ekle** (veya Siri ile kullan) → “Hey Siri, Voile’yi aç” diye kaydedin.

## Notlar

- Mikrofon izni gereklidir (Safari ilk kullanımda sorar).
- İnternet bağlantısı gerekir.
